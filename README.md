# 😂 Random Joke Generator

A simple and stylish web app that fetches random jokes from an API. If the API fails, it shows a funny fallback joke instead.

---

Example:
https://apuprna06ray.github.io/joker/

---

## 📸 Preview

A modern glassmorphism-style UI with a button that generates jokes instantly.

---

## 🛠️ Tech Stack

- HTML5
- JavaScript (Vanilla JS)
- CSS (Inline styling via JS)
- Fetch API

---

## ⚙️ Features

- 🎯 Fetch random jokes from API  
- 🔁 Fallback jokes if API fails  
- 🎨 Beautiful glassmorphism UI  
- 📱 Responsive design  
- ⚡ Fast and lightweight (no frameworks)
